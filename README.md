# Final Project of SI 507
 * Target: Scrape 2000 movie information from https://www.rottentomatoes.com/browse/dvd-streaming-all/
 * Including:
     * Movie Name
     * Genre
     * Director
     * Date in Theatre
     * Box Office
     * Tomato Meter
         * Including how many people review
     * Audience Score
         * Including how many people review
## Database tables
 * Including 3 tables
     * url_of_movie
     * basic_info_of_movie
     * tomato_meter
